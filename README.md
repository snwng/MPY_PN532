# MicroPython PN532 I2C Driver

This MicroPython module provides an interface to communicate with the PN532 NFC/RFID module over I2C. It supports authentication, reading, and writing operations on MIFARE Classic cards.

## Installation

To install the module simply use mpremote:

```
mpremote mip install github:snwng/MPY_PN532
```
